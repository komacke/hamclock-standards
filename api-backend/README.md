# HamClock Client to Backend API

This is a document of HamClock's calls to the Backend. The calls are GET method URLs.

The endpoints and arguments are documented [here](api-doc.md). It was originally generated by scraping the ohb.hamclock.app logs and generated using the [url-to-markdown.py](url-to-markdown.py) script.

A text file was similarly generated by [url-to-doc.py](url-to-doc.py) and output as [api-doc.txt](api-doc.txt).

An OpenAPI-formatted HTML file was generated from the yaml file. More information forthcoming about that technique.
